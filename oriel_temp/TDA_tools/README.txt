Oriel is an open source topological data analysis framework.  It is currently under continuous development and is highly unstable.

Dependencies:

- fastcluster
- pygraphviz
- matplotlib
- h5py *(this dependency will eventually become optional when the ability to read other types of data files is implemented)
- numpy
- sys
- time
- networkX *(this is only a dependency for the plot_matplotlib_networkX() function)
- fnmatch
- os
